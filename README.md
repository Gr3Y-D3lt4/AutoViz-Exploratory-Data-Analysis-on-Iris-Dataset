# AutoViz-Exploratory-Data-Analysis-on-Iris-Dataset
This project demonstrates Automated Exploratory Data Analysis (EDA) using the AutoViz Python library on the famous Iris dataset. The notebook showcases how AutoViz can automatically generate a wide range of informative visualizations with just a few lines of code, making data exploration faster and easier.

The project uses the Iris flower dataset loaded through Seaborn and applies AutoViz to analyze relationships, feature distributions, correlations, and class-wise comparisons.

#🚀 Features
Load the Iris dataset using Seaborn
Perform automated exploratory data analysis
Generate multiple visualizations with a single command
Analyze relationships between numerical features
Compare feature distributions across Iris species
Explore correlations among variables
Save generated plots for future use
Customize AutoViz using different parameters such as:
 depVar
 verbose
 lowess
 chart_format
 max_rows_analyzed
 max_cols_analyzed
